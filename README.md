# 项目简介

个人技术知识库，按技术领域分类组织，涵盖编程语言、Java 生态、移动端、前端、数据存储、运维部署、计算机基础等方向。

## 目录约定

采用「十位分组 + 下划线」编号体系：

| 区段 | 分类 | 示例 |
|------|------|------|
| 0x | 编程语言 | 01_Java, 02_Python, 03_Dart |
| 1x | Java 生态 | 11_SpringBoot, 12_SpringCloud |
| 2x | 移动端 | 21_Android |
| 3x | 前端 | 31_HTML-CSS-JS, 32_Vue |
| 4x | 数据存储 | 41_数据库, 42_Redis |
| 5x | 运维部署 | 51_Linux, 52_Docker, 53_Kubernetes, 54_Nginx |
| 8x | 计算机基础 | 81_数据结构, 82_算法, 83_设计模式, 84_计算机网络, 85_操作系统 |
| 9x | 其他 | 99_其他 |

文章文件名采用 `NN_标题.md` 格式，`NN` 为两位数字序号，保证自然排序；`00_` 用于分类前置概念。
